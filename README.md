### What I Build

I build practical software systems with a bias toward shipping: agentic workflows, mobile products, backend services, automation tools, and developer utilities. My work tends to sit where product ideas become working systems: connecting APIs, moving data, tightening flows, and making the machine do more of the boring parts.

- **Agentic systems:** multi-agent workflows, coding assistants, orchestration, memory, and automation surfaces.
- **Mobile products:** Kotlin, Flutter/Dart, Firebase, app flows, device QA, and release pipelines.
- **Backend systems:** APIs, service boundaries, auth flows, databases, and operational tooling.
- **Developer tools:** scripts, CLIs, dashboards, log tooling, GitHub automation, and workflow cleanup.
